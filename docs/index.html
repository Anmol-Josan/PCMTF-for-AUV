About the Simulation

Objective

The AUV must locate and track methane plumes while optimizing energy consumption using physics-informed machine learning.

Algorithm

Uses a Physics-Informed Neural Network (PINN) to learn plume dynamics and plan energy-optimal trajectories.

Energy Model

Energy consumption is quadratic with speed plus a base consumption rate. The AUV stops when energy is depleted.

Plume Dynamics

Dynamic plumes with turbulent fragmentation, drift, and noise simulate real ocean conditions.
